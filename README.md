Setup
-----
Linux (ARCH):
<br>
install: wine lib32-alsa-plugins lib32-libpulse lib32-openal
<br>
run: wine Steem.exe
<br>
Under Options: select Port > MIDI-port > <your interface> or MIDI-Through-0
<br>
Under Disk Manager: select <path to IFORMM dir>IFORMM/IFORMM

RUN
---
Start STEEM
<br>
Double Click IFORMM
<br>
type: IFORMM
